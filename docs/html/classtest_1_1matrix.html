Detailed Description

template<typename value_t, class format_t = policy::RowMajorPolicy<value_t>, class storage_t = std::vector<value_t>>
class test::matrix< value_t, format_t, storage_t >

The template class of the matrix. It is final and implements Curiously Recurring Templates Pattern. These matrices have immutable dimension and shapes will not change once created.

Template Parameters
dtypethe type that this matrix will hold.
